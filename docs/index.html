Overwatch CASC Dumper

从游戏 CASC 资源里提取英雄 / 技能 / 弹药 / state var,生成版本对应的 C++ SDK 头。零社区硬编码枚举,一切来自数据 + 可审计 CSV。

覆盖一览

核心输出

  • SDK 6 个 C++ 头文件 — namedump / abilitydump / entitydump / statevardump / weapondump / herokitdump
  • data 50 playable heroes + 1957 entities + 5594 state vars
  • CSV 3 个 overrides 文件 手工补全 / 精修

分类维度

  • Scope (var 的作用域): single_ability / single_hero / multi_hero / global
  • Usage class (var 的行为模式): sync_flag / sync_chase / sync_writable / cross_graph / temp / unused
  • Entity Slot: LMB / RMB / Shift / E / Ult / Passive / None
  • Entity Type: Turret / Projectile / Summon / Bot / Decoy / Other

数据管道

Game Update
   │
   ↓ inject keydump.dll
keydump.json (CMF/TRG 密钥)
   │
   ↓ casc_reader (dotnet)
dump_json/ (8 个 raw JSON: heroes, loadouts, abilities, entity_origins, var_usage, ...)
   │
   ↓ generate_sdk_v2.py  ←  scripts/overrides/*.csv
output/*.hpp (6 个 SDK 头) + docs/ (这个站点)
